# Archived TLS-Only Sprint

This directory preserves the superseded single persistent OpenCode server
implementation baseline that established the mandatory Java TLS proxy
architecture.

The archived specification and checklist are historical evidence. They are not
the active sprint documents and must not be updated to describe later optional
transport work.

The archived implementation and independent code audit completed, but the
runtime/container/browser gates recorded in the archived checklist remained
open. Archival does not mark those gates complete.

The active sprint documents remain at:

- `docs/single-opencode-server/SPRINT_SPEC.md`
- `docs/single-opencode-server/SPRINT_CHECKLIST.md`
